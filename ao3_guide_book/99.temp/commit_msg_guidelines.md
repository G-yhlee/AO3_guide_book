##### 커밋 메세지 가이드라인 

##### README 파일 업데이트시 아래 커밋내용 입력
> ...................................................................  
git add .  
git commit -m "update::README.md"  
git push origin main  
.....................................................................  


##### ao3_guide_book 업데이트시 아래 커밋내용 입력 + 변경사항 입력
> ...................................................................  
git add .  
git commit -m "update::ao3_guide_book::변경사항"  
git push origin main  
.....................................................................  



##### 그 외 커밋 메세지 입력

.....................................................................
